# Smart Resume Analyzer

## Overview
A full-stack application that uses AI to analyze resumes against job descriptions.

### Features
- **Resume Upload**: Upload PDF resumes.
- **AI Analysis**: Summarize resumes, extract skills, and rate job fit.
- **Cover Letter Generation**: Create tailored cover letters based on resume and job description.

### Tech Stack
- **Frontend**: Angular
- **Backend**: Spring Boot (Java)
- **AI**: OpenAI API / Local LLM
- **Tools**: Apache Tika (PDF parsing)
